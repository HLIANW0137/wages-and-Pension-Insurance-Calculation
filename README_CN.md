# Wages Calculator

[简体中文](./README_CN.md) | [English](./README_EN.md)

一款针对国内职场人的薪资计算工具，可帮你快速计算税后到手工资、五险一金扣除明细、个税缴纳金额，同时支持个人养老金收益与个税优惠测算，帮你清晰了解薪资构成与养老规划。

---

## ✨ 功能特性

- 🧮 **精准薪资计算**：支持税前工资、五险一金、个税全维度计算，精准算出税后到手工资，覆盖最新个税政策。

- 🏙️ **多城市政策适配**：支持自定义缴费基数、个人 / 单位缴费比例，可适配全国不同城市的社保公积金政策。

- 🧓 **个人养老金测算**：支持个人养老金个税优惠测算、长期投资收益预估，帮你做好养老规划。

- 📋 **专项附加扣除**：完整支持子女教育、赡养老人、住房贷款 / 租房、继续教育等全部个税专项附加扣除项。

- 📊 **明细化结果展示**：完整展示每一项扣除明细，社保 / 公积金 / 个税 / 专项扣除一目了然，单位与个人缴费部分分开展示。

- 🔒 **本地隐私计算**：所有计算逻辑均在前端本地完成，无需上传任何个人数据，隐私安全有保障。

---

- **前端**: React 19, Vite 6.2, Tailwind CSS v4, Framer Motion, Lucide Icons

## 🚀 快速开始

### 前置要求

- Node.js（推荐 v18 及以上版本）

- npm 或 yarn

### 安装步骤

1. **克隆仓库:**

    ```bash
    
    git clone https://github.com/HLIANW0137/wages-and-Pension-Insurance-Calculation.git
    cd wages-and-Pension-Insurance-Calculation
    ```

2. **安装依赖:**

    ```bash
    
    npm install
    ```

3. **启动开发服务:**

    ```bash
    
    npm run dev
    ```

4. **打开浏览器:**
访问 [http://localhost:3000](http://localhost:3000) 即可使用。

## 📖 使用说明

1. **输入基础薪资**：在输入框中填写你的税前月薪。

2. **配置缴费规则**：选择你所在的城市，自动加载当地的社保公积金默认缴费比例，也可以根据实际情况手动调整。

3. **填写专项扣除**：根据你的实际情况，填写个税专项附加扣除项，包括子女教育、赡养老人、住房贷款等。

4. **查看计算结果**：一键生成计算结果，查看到手工资、各项扣除明细，以及个人养老金的测算结果。

## 🔧 自定义与调整

本项目设计为高度可定制，你可以根据需求调整适配最新政策：

### 1. 调整城市默认配置

如果你想新增或修改城市的默认缴费比例，可以修改配置文件中的城市配置项，快速适配你所在城市的最新政策。

### 2. 更新计算规则

如果个税 / 社保政策更新，你可以修改 `src/utils/calculation.ts` 中的计算逻辑，快速适配最新的政策规则。

### 3. 调整 UI 样式

界面使用 Tailwind CSS 开发，你可以修改 `src/components/` 下的组件，调整界面的样式、布局，适配你的个性化需求。

### 4. 扩展功能

你可以轻松扩展更多实用功能，比如：

- 年终奖个税计算（单独计税 / 合并计税对比）

- 社保缴费年限与养老金领取测算

- 公积金贷款额度测算

- 年度收入汇算清缴测算
